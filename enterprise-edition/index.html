React-Admin Enterprise Edition already contains the following modules:

  • ra-preferences: Persist user preferences (language, ui, filters, displayed columns, etc) in local storage.
  • ra-realtime: Live notifications, updates and semaphores taking advantage of real time backends.
  • ra-breadcrumbs: Hierarchical navigation (menu, breadcrumbs)
  • ra-many-to-many: Visualize and edit many-to-many relationships using an association table
  • ra-editable-datagrid: Edit data directly in the list, for better productivity.
  • ra-tree: Edit and visualize tree structures. Reorganize by drag and drop. Adapts to any data structure on the backend (parent_id, children, nested sets, etc).
  • ra-tour: Guided tours for react-admin applications.
  • ra-markdown: Read Markdown data, and edit it using a WYSIWYG editor in your admin

Each of these modules contains both hooks and components, in order to ease the integration with your UI. You get full access to the source, source maps and documentation. Private modules follow the same code and documentation quality rules as the open-source packages. The Enterprise Edition also include a storybook and a dedicated demo, where you can learn how to use the private modules by example.
